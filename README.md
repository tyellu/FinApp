Description of Web App:
	Finance App that allows users to create and manage a virtual trading portfolio and allows users
	to simulate trades.

		- Login to account
	- API Integration
		- Portfolio
	- Simple Dashboard
		- Portfolio Overview
		- Portfolio Drill down
		- Account Overview

		- Watchlist
		- Market Overview
	- Simple Dashboard
		- Market Overview
		- Room
